离散数学: 1.2 Applications of Propositional Logic

发表于 2017-04-24

从名字可以看出，这一节是对上一节知识的应用。在这一节中，介绍了在搜索引擎中如何利用AND、OR等进行精确查找所需内容，还有一些逻辑谜题、电路实现逻辑运算的内容。整节比较简单易懂。

[TOC]

1. Introduction

2. Translating English Sentence

把自然语言翻译为逻辑语言的练习。自然语言具有不精确、磨棱两可、省略许多背景信息的特点，而逻辑语言则要求准确无误，所以在翻译过程中一定要认真，理清关系。

3. System Specifications

这个小节是说当几个命题组成一个”system”的时候，必须保持“一致性”，不能互相矛盾。

4. Boolean Searches

  • AND去搜索两个目标都存在的网页。
  • OR去搜索只要有两者之一或者两者都有的网页。
  • NOT去搜索两者剔除其中一者的网页。

5. Logic Puzzles

介绍了几个运用逻辑的小问题，类似于脑筋急转弯。

6. Logic Circuits

简单介绍了逻辑运算的电路实现形式，分别有“非门”、“或门”、“与门”三种。
